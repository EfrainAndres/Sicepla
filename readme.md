# Interventorias

### Instalación

El proyecto esta desarrollado en [Laravel 5.4](https://laravel.com/docs/5.4/)

Despues de clonar el proyecto crear una copia del archivo .env.example y ejecutar.

```sh
$ composer install
$ php artisan key:generate
$ php artisan migrate
$ php artisan db:seed
```
